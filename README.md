# MSU-ZS ROTC Inventory System

A comprehensive inventory management system for the MSU-ZS ROTC Unit, built with [Next.js](https://nextjs.org).

## Live Demo

The application is deployed on Vercel. Check the latest deployment URL for the live version.

## Tech Stack

- **Framework:** Next.js 16
- **UI:** React 19, Tailwind CSS 4, shadcn/ui
- **Database:** Supabase (PostgreSQL)
- **Auth:** Supabase Auth
- **Language:** TypeScript

## Getting Started (Development)

```bash
npm run dev
```

## Environment Variables

Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials.

## Build & Deploy

```bash
npm run build    # Production build
npm run lint     # Lint check
```

Deploy to Vercel by connecting your GitHub repository.
